# CAD

Put Fusion 360 / SolidWorks / FreeCAD source files here alongside exported
STEP and STL files. Commit source files (.f3d, .sldprt, .FCStd); avoid
committing backup files (the .gitignore already filters `*.bak`).

Suggested subfolders:
- `assemblies/` — full-arm assembly files
- `parts/` — individual part models
- `exports/` — STEP/STL exports for slicing or sharing
